# Black Hole ASCII Simulation

Реализация симуляции черной дыры с аккреционным диском в формате живого ASCII art.

## Математическая основа
- Метрика Шварцшильда для искривления пространства-времени(https://www.mi-ras.ru/noc/13_14/2/lectures.22.04.14.pdf)
- Трассировка лучей с учетом гравитационного линзирования(https://dxdy.ru/topic96508.html)
- Релятивистское доплеровское смещение(https://physics42.ru/tutorials/relyativistskaya-fizika/doplerovskiy-effekt-v-relyativistskom-sluchae/)
- Моделирование аккреционного диска(https://physics42.ru/tutorials/fizika-chyernykh-dyr/teoriya-akkretsionnykh-diskov/)
