# Black Hole ASCII Simulation

Реализация симуляции черной дыры с аккреционным диском в формате живого ASCII art.

## Математическая основа
- Метрика Шварцшильда для искривления пространства-времени
- Трассировка лучей с учетом гравитационного линзирования
- Релятивистское доплеровское смещение
- Моделирование аккреционного диска
